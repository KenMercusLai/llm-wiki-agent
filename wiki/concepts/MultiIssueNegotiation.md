# Multi-Issue Negotiation / 多议题谈判

Multi-issue negotiation is the practice of putting several interests on the table at once rather than bargaining over one point at a time. [[106-bie-zai-haipa-chongtu-kuai-lai-xuexi-tanpan-shu-742539645]] uses television licensing examples to show how price, number of broadcasts, future show rights, and risk-sharing can be traded against one another.

The concept matters because parties often value issues differently. If one side can gain a lot from an issue that costs the other side little, the parties can create new value rather than merely split a fixed pie. The episode also notes that discussing multiple issues reveals priorities because each side's ranking becomes visible.

## Key Claims
- One-issue bargaining encourages each side to treat the current point as maximally important.
- Bundled proposals make it easier to trade low-priority concessions for high-priority gains.
- Multi-issue negotiation can expand the effective [[ZoneOfPossibleAgreement]].
- Priority ranking itself is useful information for [[InterestBasedNegotiation]].
