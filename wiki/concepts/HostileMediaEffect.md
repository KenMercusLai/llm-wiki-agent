# Hostile Media Effect / 敌意媒体效应

Hostile media effect is the pattern where opposing groups interpret the same nominally neutral coverage as biased against their own side. [[123-renxing-shiyan-dangxin-ni-ziji-779479569]] explains it through sports-fan perception, election-reporting complaints, and a study of television news about the Arab-Israeli conflict.

The episode's practical point is that political knowledge and confidence do not necessarily reduce perceived bias. Strong identification can make the same clip look like enemy evidence to both camps.

## Key Claims
- Group identity can change what counts as neutral.
- People can sincerely perceive the same media object in incompatible ways.
- Strong prior positions can make balanced reporting feel like betrayal.
- Bias correction requires more than telling people to be objective; it requires active standpoint testing through [[ConsiderTheOppositeBiasCorrection]].
