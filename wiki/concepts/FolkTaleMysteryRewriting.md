# 民间故事推理改写 / Folk-Tale Mystery Rewriting

民间故事推理改写 is the adaptation pattern added by [[99-henjiu-henjiu-yiqian-yaoguai-ye-tuili-a-730851220]]. In this mode, a familiar folk tale is not only retold or demystified; it is converted into a mystery system where inherited motifs become rules, clues, suspects, false explanations, or hidden identities.

The episode's key example is [[MukashiMukashiAruTokoroNiShitaiGaArimashita|《很久很久以前，在某个地方有一具尸体》]]. [[IssunBoshi|一寸法师]] contributes size change, ogre digestion, a magic mallet, and a tiny-body premise. [[SaruKaniGassen|猿蟹合战]] contributes revenge roles, animal society, and shapeshifting. The stories remain recognizable, but the inherited wonder is reorganized into [[SettingBasedMystery|设定系推理]].

This differs from pure source-history inquiry. Pages such as [[StoryMotifTransmission]], [[EvidenceBoundFolkloreInquiry]], and [[MythicSourceLayering]] ask how motifs travel or accumulate. Folk-tale mystery rewriting asks a different question: once the motif is accepted as part of a fictional world, what deduction does it make possible?

## Key Claims
- Familiar tales can function as shared rulebooks for mystery readers.
- Rewriting can preserve magical premises instead of explaining them away.
- A motif becomes a clue when it has operational consequences: size, transformation, taboo, repetition, rescue, or revenge changes what can happen.
- The form can make old tales feel newly sharp by changing the moral angle: heroes can become suspects, victims can deceive, and comic helpers can become coded participants.
- Motif familiarity lowers exposition cost but raises fairness pressure; the mystery still has to state which version or rule it is using.
