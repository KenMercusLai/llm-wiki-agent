# Everything Agent

Everything Agent is the host's label for an investment direction where agents enter many parts of white-collar work. The first source highlights agent infrastructure needs such as sandboxing, memory, communication networks, payment, and making products discoverable and usable by agents. [[agent-yuannian-di-500-tian-shenme-zai-xiaoshi-shenme-zai-dansheng-weishenme-women-bugai-zai-touzi-gui-siwei-de-ruanjian-lhwdxfpke3bmamjk4e6knk-5sn-b]] develops the same direction through [[AgenticEconomy]], where agents may exchange information and value across new infrastructure.

## Investment Logic
- Agents will need more than model intelligence; they need operating environments.
- Future products may have to serve both humans and agents acting for humans.
- [[WeChat]] is discussed as a possible high-context platform where an agent could become especially powerful in China.
- [[HeadlessSoftware]] and [[AgentFacingInterfaces]] describe how products become usable by agents rather than only humans.

- [[AgenticWorkflow]] — user-level version of the same shift.

- [[AgenticEconomy]] — infrastructure and economics version of the thesis.
