# AI Storage Supercycle

AI Storage Supercycle is the source's name for the memory and storage upcycle driven by AI infrastructure demand in [[cunchu-sanjutou-po-wanyi-shizhi-cunchu-chaoji-zhouqi-heshi-neng-jianding-s10e13-c47ff830-8cb5-4e58-b7d7-1a04e4e5a4c1]]. Unlike older PC or smartphone replacement cycles, this one combines demand for DRAM, [[HighBandwidthMemory]], and NAND at the same time.

The source argues that the cycle may last longer because AI server customers are less price-sensitive, suppliers are more capital-disciplined after the last downturn, and customers are using [[MemoryCapacityLockIn]] to secure future supply. It still warns that the supercycle is not a permanent escape from [[StorageIndustryCyclicality]].

## Key Claims
- AI demand changes the memory cycle by adding HBM and inference-driven NAND demand to ordinary DRAM and server demand.
- Consumer electronics can be squeezed because server buyers can absorb higher memory prices than low-margin phones or PCs.
- Supply may remain tight into 2027-2028 in the source's account because capacity additions lag demand and supplier expansion is cautious.
- The supercycle is a weaker or longer cycle, not the end of cycles.
