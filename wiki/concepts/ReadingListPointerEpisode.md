# Reading List Pointer Episode

Reading list pointer episode is the format exemplified by [[kaitianchuang-zhi-lu-jian-tuijian-ji-ben-shu-726703080]]. Instead of developing one book at length, the host uses a short "开天窗" slot to route listeners toward an off-platform exclusive episode, preview current reading, and mark possible future topics without making firm programming promises.

## Key Claims
- The format is useful for the wiki because it records leads, not settled arguments.
- Book pages created from this kind of source should stay thin and source-scoped until fuller discussions are ingested.
- The format still matters: it shows how [[MihuanChishu|蜜獾吃书]] builds continuity among reading life, platform routing, future episodes, and informal audience relationship.
- It is adjacent to [[ReadingAsLifeExperience]] because the host's reading life, mood, schedule, and curiosity shape the episode as much as the books' formal arguments.
