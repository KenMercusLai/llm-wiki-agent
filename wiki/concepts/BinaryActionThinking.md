# Binary Action Thinking

Binary action thinking is the episode's method for reducing perfectionist over-scoring. In [[129-ruhe-chengwei-zhenzheng-de-buwanmei-zhuyizhe-hunshen-shi-jiner-799220151]], [[QinZong|秦总]] proposes treating a task as 0 or 1: if the action happened, it counts as 1; if it did not happen, it remains 0.

The point is not to deny quality forever. The point is to stop grading posture, speed, wording, clothing, pauses, and every other detail before action has a chance to exist. For perfectionists trained by [[RedPenLogic]], binary action thinking temporarily protects the first completed version from being crushed by evaluation.

## Key Claims
- Counting completion before quality can weaken the fear that every attempt must be scored immediately.
- The method is useful for actions where non-starting is the main failure mode.
- Complex tasks can be decomposed into multiple binary parts so progress becomes visible.
- Binary thinking should be temporary: later revision can improve quality after the 0-to-1 step exists.
