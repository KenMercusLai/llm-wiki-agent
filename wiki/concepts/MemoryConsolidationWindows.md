# Memory Consolidation Windows / 记忆巩固窗口

Memory consolidation windows are the early periods after learning when [[143-yiwang-de-naoli-jixing-buhao-ke-zha-zheng-a-869667634]] argues review and repeated stimulation matter most. The episode uses the Ebbinghaus forgetting curve as a practical heuristic: much forgetting happens quickly, so the first tens of minutes and the first 24 hours are especially valuable for restating, summarizing, and reactivating material.

The concept is not "study harder forever." It is the complement to [[ForgettingAsCognitiveFunction]]: if forgetting is normal, learners should decide which knowledge deserves deliberate reinforcement and which material can be allowed to fade.

## Key Claims
- Review works best when it happens before a memory trace has already weakened heavily.
- Repetition can be stronger when paired with summarizing, organizing, and adding sensory or situational cues.
- The episode compares hippocampus and cortex to temporary and longer-term storage, using that metaphor to explain why some information needs repeated transfer.
- Sleep after review can protect consolidation by reducing competing stimulation.
- Strong new stimulation immediately after last-minute review may interfere with keeping fragile material active.
- A learning plan should preserve attention and rest, not only increase hours.
- The source uses Ebbinghaus as a practical study frame; it does not claim one curve covers every learner, subject, or assessment type.
