# Semantic Parsing

Semantic parsing is described in [[139-agent-de-zongshu-he-su-yu-liao-agent-jishushi-openclaw-moment-bianjie-de-xiaomi-he-shehui-de-fushe-luffrgudeiighqxam49tfqci63no]] as the NLP route that translates natural language into machine-readable semantic representations. Those representations can connect human intent to databases, knowledge graphs, websites, or executable tools.

[[SuYu]] frames semantic parsing as complementary to neural agents: neural agents went deep in constrained environments, while semantic parsing broadened the possible action space by letting language address structured systems. His own PhD work in the area is presented as a precursor to his later [[LanguageAgent]] and [[ComputerUseAgent]] work.

## Key Claims
- Semantic parsing tries to let machines understand human instructions without forcing people to think like software systems.
- It expands agent action space by connecting language to structured back ends and executable environments.
- It prefigures current [[AgentFacingInterfaces]] work because both ask how human intent becomes reliable machine action.

## Connections
- [[SuYu]] — researcher whose path moved from semantic parsing toward language agents.
- [[LanguageAgent]] — later paradigm that absorbs language-to-action capabilities into LLM-based systems.
- [[AgentFacingInterfaces]] and [[AgentHarness]] — modern infrastructure layers that continue the language-to-action problem.
- [[UniversalDigitalAgent]] — convergence target that requires broad action spaces across software environments.
