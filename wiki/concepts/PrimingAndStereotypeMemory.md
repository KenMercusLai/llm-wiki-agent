# Priming And Stereotype Memory / 启动效应与刻板印象记忆

Priming and stereotype memory is the source's pattern where earlier words, labels, or identity cues shape later evaluation and recall. [[123-renxing-shiyan-dangxin-ni-ziji-779479569]] describes a word-search experiment in which positive or negative trait words altered evaluation of the same ambiguous adventurous behavior, and a later-memory experiment where occupational labels shaped what participants remembered about a woman.

The concept matters because bias does not always appear as explicit hostility. It can enter before judgment through available words, then reappear later as apparently remembered detail.

## Key Claims
- Ambiguous behavior can look brave or reckless depending on prior conceptual activation.
- Stereotypes can reshape memory, not only immediate opinion.
- Memory and stereotype can reinforce each other when the altered recollection feels like direct evidence.
- Limited mental resources make simplification tempting, but simplification can become social misrecognition.
