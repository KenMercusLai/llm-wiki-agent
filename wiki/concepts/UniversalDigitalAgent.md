# Universal Digital Agent

Universal digital agent is [[SuYu]]'s future target in [[139-agent-de-zongshu-he-su-yu-liao-agent-jishushi-openclaw-moment-bianjie-de-xiaomi-he-shehui-de-fushe-luffrgudeiighqxam49tfqci63no]]: an agent that can operate across the digital world rather than being confined to a web, desktop, mobile, coding, or browser-use niche.

The concept is not presented as a claim that one product already controls every entry point. It is a direction implied by boundary dissolution: coding can manipulate many digital systems, [[AgentFacingInterfaces]] can expose callable software capabilities, GUI remains necessary for human-visible systems, and [[AgentHarness]] infrastructure can coordinate tools, memory, context, permissions, and feedback.

## Key Claims
- Current agent labels are provisional because real digital tasks cut across websites, local apps, code, files, messages, calendars, and APIs.
- [[ComputerUseAgent]] is a key step, but the long-term user need is task completion across multiple digital surfaces.
- [[ContinualLearning]] and [[SpecializedIntelligence]] are required because universal reach without domain learning produces unreliable generic automation.
- Human trust still needs reviewable surfaces, so universal agents should not be read as the disappearance of GUI or audit.

## Connections
- [[LanguageAgent]] and [[ComputerUseAgent]] — technical base and transitional category.
- [[OpenClawMoment]] — interaction shock that made the direction concrete.
- [[AgentHarness]], [[AgentFacingInterfaces]], and [[AgentPermissionBoundaries]] — infrastructure needed to operate across digital systems.
- [[WorldModels]], [[ContinualLearning]], and [[SpecializedIntelligence]] — learning requirements for reliable domain work.
