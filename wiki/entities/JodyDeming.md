# Jody Deming

Jody Deming is the marine microbiologist referenced in [[melody-jue-ocean-memory-wxagwid1oeg]] for reframing Arctic sea ice as a porous habitat where microbes can become memory agents. In [[MelodyJue]]'s account, Deming's work helps shift ice from a passive climate record into a living and politically interpreted medium.

The source uses Deming's example to link microbial behavior with [[OceanMemory]] and [[EcologicalMemory]]. Memory here is not only stored evidence; it can also be anticipation, recall, and altered response after prior exposure.
