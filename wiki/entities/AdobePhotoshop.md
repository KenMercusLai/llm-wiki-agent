# Adobe Photoshop

Adobe Photoshop is referenced in [[263-sora-si-le-adobe-die-le-meitu-he-qu-he-cong-lgjmyveooc8wpzr0yviggvzvdyfs]] as one of [[Adobe]]'s professional creative surfaces where AI features can be embedded into an existing workflow. The episode contrasts this with [[Meitu]]'s more flexible application-company posture and [[ModelContainerStrategy]].

For the wiki, Photoshop is a useful example of an existing tool with strong workflow habit. The source's tension is that workflow ownership can support [[AIApplicationLayerMoat]], but AI features still have to justify [[AIInferenceCostStructure]] through user value and pricing.
