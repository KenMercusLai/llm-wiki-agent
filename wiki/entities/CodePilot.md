# Code Pilot

Code Pilot is described by [[CangShifu]] in [[agent-yuannian-di-500-tian-shenme-zai-xiaoshi-shenme-zai-dansheng-weishenme-women-bugai-zai-touzi-gui-siwei-de-ruanjian-lhwdxfpke3bmamjk4e6knk-5sn-b]] as a coding agent similar in ambition to [[Codex]] or [[ClaudeCode]]. Its stated direction is openness, localization, and keeping skill, memory, CLI, and harness components under more local user control.

[[1-ren-gongsi-kang-5-ge-ren-de-huo-hai-yao-guan-50-ge-agents-s10e18-e3a21dde-0bba-4ec2-bf12-5043500ae5c6]] adds the commercialization boundary. Cang Shifu says building and releasing CodePilot-like software is not only an engineering problem; once the project becomes more commercial, compliance, finance, KYC-like processes, user trust, and ongoing operations take much more human time.
