# Claude Tag

Claude Tag is the [[Anthropic]] team-collaboration surface described in [[ai-jibao-26q2-cong-coding-dao-rsi-qiangzhe-yu-qiang-de-weilai-1-171-1]]. The source says users can mention Claude in [[Slack]], assign work, and receive results back in the channel, turning Claude from a private assistant into a visible team coworker.

The source treats Claude Tag as a product-shape change for [[AgentHarness]] work. Team agents need channel context, task ownership, permission boundaries, and social visibility; otherwise a strong model remains a private chat tool rather than part of a group's operating rhythm.
