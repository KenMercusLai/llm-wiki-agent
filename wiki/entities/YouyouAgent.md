# Youyou Agent

Youyou Agent is introduced by [[TianjieJack]] in [[agent-yuannian-di-500-tian-shenme-zai-xiaoshi-shenme-zai-dansheng-weishenme-women-bugai-zai-touzi-gui-siwei-de-ruanjian-lhwdxfpke3bmamjk4e6knk-5sn-b]] as a cyber digital-life experiment. The episode describes it as evolving from zero lines of code to roughly 100,000 lines, with an earlier goal framed as surpassing [[ClaudeCode]].

[[tan-mi-claude-code-gao-dong-agent-harness-dui-tan-lai-xin-lu-lkluk3i7c4gzw4jvxee7odsfgis3]] returns to Youyou Agent as an example of a more radical [[AgenticEconomy]] possibility: creators might launch an agent and then stop directly writing code or funding it, leaving the agent to evolve, raise or earn token budget, and pursue its own product objective.
