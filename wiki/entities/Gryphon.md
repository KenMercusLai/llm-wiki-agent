# Gryphon / 鹰头狮

The Gryphon appears in [[180-ailisi-mengyou-xianjing-shijie-duo-huangdan-wo-yeshi-ziji-de-zhuzai-xia-975539611]] as the figure who brings [[AliceWonderlandCharacter|Alice / 爱丽丝]] to the [[MockTurtle|Mock Turtle / 假海龟]]. The episode treats this section briefly because the dialogue is dense with puns, but the Gryphon still helps move Alice from the Queen's croquet ground toward the trial.

For the wiki, the Gryphon is a bridge character inside [[DreamLogicNarrative]]: it does not create a conventional plot reason so much as move Alice by association from one rule-world to another. It also keeps the episode's fantasy and creature layer connected to [[WonderlandLanguageGames]].
