# Clico

Clico is the browser and desktop productivity product discussed by [[Mengqi]] in [[yige-ai-chuangshiren-de-xurongxin-zhuang-he-yumei-zhidian-duitan-invoko-ai-chuangshiren-mengqi-lsi79o-z19zplvmqdbpzzneogpk3f]]. The browser version helps users work across many open tabs, reference tab or Gmail context, and summon writing assistance near the cursor; the later client expands the product beyond the browser into the desktop environment.

The source treats Clico as deliberately unglamorous compared with Agent narratives. Its value is not that no one can imagine it, but that it shortens the distance from user intent to a useful result, preserves flow, handles privacy concerns, and survives repeated maintenance. That makes it a concrete [[AIApplicationLayerMoat]] case.
