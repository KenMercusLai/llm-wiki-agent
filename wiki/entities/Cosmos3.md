# Cosmos 3

Cosmos 3 is discussed in [[jushen-jibao-26q2-shijie-moxing-dafeng-buting-he-buxiang-bei-tie-biaoqian-de-ren-1-170-1]] as [[Nvidia]]'s Q2 2026 marker for productized [[WorldModels]]. [[ChenZhePeter]] describes it as a more open, product-level omni-world-model release with Super, Nano, and Edge variants for different deployment contexts.

The episode says Cosmos 3 can handle and generate multiple modalities, including text, images, video, sound, and action, and uses a Mixer of Transformers architecture combining autoregressive and diffusion transformer components. Its significance in the wiki is not only model capability, but the fact that a major infrastructure company can release a world-model stack that supports [[WorldActionModels]], robot simulation, and [[WorldModelVLAFusion]] without needing the model itself to be the whole robot policy.
