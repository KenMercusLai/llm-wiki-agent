# GitHub

GitHub appears in [[tsr-ronconway-part4-v2-tsr-ronconway-part4-v2]] as [[SeanFanning]]'s reciprocal gift to [[RonConway]] after years of difficult [[Napster]] and [[Snowcap]] work. At a dinner in San Francisco, Fanning told Conway that GitHub was special because engineers stored their code there and the company could not be allowed to go out of business.

Conway says he met the GitHub founders, spent about a year educating them about venture capital, and helped syndicate a funding round. In this source, GitHub is less a developer-tool product analysis than a case in [[FounderInvestorLearning]] and [[FounderFriendlyInvestorSupport]]: a long founder relationship created access to a high-conviction infrastructure company.
