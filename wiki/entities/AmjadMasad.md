# Amjad Masad

[[tsr-s3-replit-v2-tsr-s3-replit-v2]] makes Amjad Masad a full founder-origin case for [[Replit]]. Amjad says he started the earliest browser-based programming work in Jordan because language setup was too hard and he did not have easy laptop access. He first put toy languages such as Lisp in the browser, then compiled Python from C to JavaScript, open sourced the work, and posted it to [[HackerNews]] in 2011.

The episode traces Amjad through [[Codecademy]], where his open-source Replit work was used by a YC company, and [[Facebook]], where he joined the early [[ReactNative]] team for a harder technical challenge. Those detours made him cautious about startup stress, but he and [[HayaOdeh]] eventually chose to build Replit because the mission of making programming accessible mattered enough to keep going.

Amjad's 2015 or 2016 Replit "master plan" links [[AccessibleProgrammingPlatform]], AI-assisted coding, and [[ConsumerGradeDeveloperInfrastructure]]: start with programming education, use product data to train AI help, then become a one-stop shop for building, hosting, deploying, scaling, and running software businesses. The source also makes him a security and infrastructure founder, because Replit's simple interface depends on hard work around hosted execution, package management, and abuse prevention.

Amjad appears in [[tsr-ycoffsite-pg-audioonly-final-tsr-ycoffsite-pg-audioonly-final]] as the Replit figure whose early AI-coding demo helped [[PaulGraham]] understand [[VibeCoding]]. Graham recalls Amjad telling him not to look too closely at the generated code because the English instruction was becoming the source-level artifact and the emitted code was more like object code.
