# Amjad Masad

Amjad Masad appears in [[tsr-ycoffsite-pg-audioonly-final-tsr-ycoffsite-pg-audioonly-final]] as the [[Replit]] figure whose early AI-coding demo helped [[PaulGraham]] understand [[VibeCoding]]. Graham recalls Amjad telling him not to look too closely at the generated code because the English instruction was becoming the source-level artifact and the emitted code was more like object code.

The source uses Amjad less as a biographical subject than as a product-philosophy signal. His demo gives the wiki a concise formulation for AI-assisted programming: if the model can reliably translate intent into working software, then the human's main work shifts toward specification, testing, acceptance, and product judgment.
