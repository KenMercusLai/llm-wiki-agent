# 许慎

许慎 is discussed in [[57-shuojie-hanzi-shiniaopi-weisha-shi-shi-zitou-642970319]] as the authorial figure behind [[ShuowenJiezi|《说文解字》]]. The episode presents him with respect but not deference: his dictionary is crucial, yet some explanations can be corrected because modern scholars have more excavated evidence.

The source uses Xu Shen's limits to sharpen [[ChineseCharacterEvidenceDiscipline]]. Authority, tradition, and elegant explanation are not enough; character interpretation still has to account for oracle-bone forms, bronze inscriptions, later script changes, and possible scribal distortion.
