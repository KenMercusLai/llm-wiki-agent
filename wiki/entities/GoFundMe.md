# GoFundMe

GoFundMe appears in [[socialradarss2-billclerico-final]] as one of [[WePay]]'s important early API customers. [[BillClerico]] says GoFundMe was still a very small company when it began using WePay's payments infrastructure around 2012 or 2013.

The source uses GoFundMe to show why WePay had to mature from manual payments operations into scalable infrastructure. Growth from API customers made transaction review impossible to handle by hand and exposed [[EarlyFintechFraudControls]] as a survival problem.
