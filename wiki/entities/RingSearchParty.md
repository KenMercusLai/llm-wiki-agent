# Ring Search Party

Ring Search Party is the AI-powered [[Ring]] feature discussed in [[tech-20260220-0220-mp-tech-pod-128-tech-20260220-0220-mp-tech-pod-128]]. The episode describes the public pitch as a lost-dog tool that can use connected Ring cameras to help reunite pets with families.

The feature matters because the episode reads it as a boundary case for [[ConsumerCameraSurveillance]]. A pet-finding workflow is emotionally appealing and user-controlled, but the same networked cameras and recognition logic can also be imagined as neighborhood-scale monitoring, especially when combined with law-enforcement relationships and a planned [[FlockSafety]] partnership.
