# Mycontext

Mycontext is the open-source context-capture project discussed as [[AirJelly]]'s predecessor in [[openclaw-zhihou-shui-jiang-dingyi-zhudongshi-ai-de-xin-zhanchang-duitan-airjelly-huang-bote-lplswo8r829akxwgyurfkojelku6]]. [[HuangBote]] says he worked on it at [[ByteDance]] and that it used periodic screenshots to capture and analyze user context.

The source says Mycontext had more than 5,000 GitHub stars and helped form Huang's long-running interest in context-aware AI. AirJelly changes the product direction by moving from fixed-interval capture toward event- and intent-triggered [[OSLevelContext]], then adding memory and execution ability.
