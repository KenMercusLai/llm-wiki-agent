# Robby Wokbot

Robby Wokbot is the automated wok machine in [[chef-vs-robot]], used by [[KennyPoon]] at [[Instafoods]]. The [[PlanetMoney]] episode describes Robby as a six-foot-tall metal machine with a touchscreen, heated spinning basket, ingredient prompts, sauce control, and self-washing.

Robby matters because it is partial automation. It does not eliminate all kitchen labor: a human still adds ingredients and follows the screen. It changes which skills are scarce by moving some work from trained wok technique toward machine operation, consistency, throughput, and maintenance.
