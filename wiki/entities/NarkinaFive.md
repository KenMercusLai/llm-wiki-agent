# Narkina Five

Narkina Five is the prison-labor setting analyzed in [[42-anduo-fengqi-yu-qingping-zhi-mo-616391638]]. The episode treats it as [[Andor|《安多》 / Andor]]'s most explicitly dystopian space: clean rooms, food, group routines, electric floors, productivity competition, and release hopes are all used to make prisoners useful to [[GalacticEmpire]].

The revolt begins when [[CassianAndor]], [[KinoLoy]], and others learn that release is a lie and that prisoners are simply recycled elsewhere. The source therefore uses Narkina Five to build [[CarceralLaborGovernance]] and [[ManufacturedPrisonerDilemma]].
