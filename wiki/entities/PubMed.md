# PubMed

PubMed appears in [[no-206-jiansuo-songyao-kanbing-hulianwang-yiliao-zhexie-nian-zhongguo-hulianwang-gushi-22-991273500]] as one of the first internet tools that materially helped doctors and medical students. The episode contrasts free online MEDLINE retrieval with earlier CD-ROM search and manual journal lookup, treating PubMed as a global access point for medical knowledge.

For the China internet-healthcare story, PubMed matters because it shaped [[LiTiantian]]'s early work on [[DXY]]. The source uses that path to show that [[InternetHealthcare]] first developed through professional information access before large consumer-facing services arrived.
