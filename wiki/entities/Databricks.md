# Databricks

Databricks appears in [[tsr-ycoffsite-kasishgupta-v1-audioonly-tsr-ycoffsite-kasishgupta-v1-audioonly]] as another enterprise data system that [[Hightouch]] customers used as a source for marketing and sales workflows. [[KashishGupta]] groups it with [[Snowflake]] when explaining the gap between having customer data and actually using that data in production go-to-market systems.

The source makes Databricks part of the [[EnterpriseDataActivation]] branch. For Hightouch, the enterprise opportunity was not to replace the customer's database but to connect governed data from that database into operational marketing tools without taking data ownership away from the customer.
