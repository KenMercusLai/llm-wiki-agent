# Learn Claude Code

Learn Claude Code is [[ShareAI]]'s learning project for studying [[ClaudeCode]] as a practical [[AgentHarness]] sample. In [[tan-mi-claude-code-gao-dong-agent-harness-dui-tan-lai-xin-lu-lkluk3i7c4gzw4jvxee7odsfgis3]], [[LaiXinlu]] says the project started roughly nine months before the episode and uses Claude Code's design to teach agent tools, memory, context compression, skills, and handoff patterns.

## Connections
- [[ClaudeCode]] — main system being studied.
- [[LaiXinlu]] and [[ShareAI]] — person and company behind the project.
- [[AgentHarness]] — conceptual frame the project tries to make concrete.
- [[AISkills]], [[PersistentAgentMemory]], and [[ContextEngineering]] — mechanisms highlighted through Claude Code analysis.
