# Axiom

Axiom is the [[AIForMath]] company founded by [[HongLetong]] and discussed in [[137-dui-hong-letong-de-4-xiaoshi-fangtan-ai-for-math-ba-shuxue-biancheng-lean-shuxue-tianshu-zhong-de-zhengming-zhijue-bei-chuangzao-yu-bei-faxian-de-lha-faiwxtget0qmbcosts3cb5vb]]. The source presents the company as a deep-tech systems effort focused on formal mathematical reasoning rather than a generic model company.

## Source Position
- Axiom's core product direction is [[AxiomProver]], an AI theorem-proving system built around [[LeanTheoremProver]]-style verification.
- The company is described as needing AI, reinforcement learning, agents, code generation, compiler, Lean, Mathlib, metaprogramming, and pure-math talent.
- Hong says Axiom had to build auxiliary tools because Lean data is scarce, the language is brittle, and verification can be slow.
- The episode claims Axiom raised a large seed round and then an A round, while emphasizing that investor interest depended on team execution and system progress rather than a simple model demo.
- [[FormalVerification]] is framed as Axiom's best first market because chips, software, and complex systems already need stronger proof than finite tests can provide.
