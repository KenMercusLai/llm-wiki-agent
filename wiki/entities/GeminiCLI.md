# Gemini CLI

Gemini CLI is discussed in [[ep108-vibe-coding-da-dizhen-cursor-dingjia-zhengyi-windsurf-shougou-fengbo-moxing-changshang-qin-erzi-men-you-jiang-ruhe-jinchang-lqn-icq1xqgk7xxxxzrpunj4fan]] as a Google-linked command-line coding tool using [[Gemini]]. The source highlights two practical points: a large context window can make "load the full source" approaches effective, and users can spend high-quality model turns on planning before cheaper or downgraded turns execute against a clear plan.

The episode contrasts Gemini CLI with [[Cursor]] by suggesting that direct model-provider tools may sometimes preserve more complete context than editor products that chunk, index, or retrieve code to manage cost.
