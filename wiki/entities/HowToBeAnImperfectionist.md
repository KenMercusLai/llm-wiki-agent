# 《如何成为不完美主义者》 / How to Be an Imperfectionist

《如何成为不完美主义者》 is the book discussed in [[129-ruhe-chengwei-zhenzheng-de-buwanmei-zhuyizhe-hunshen-shi-jiner-799220151]]. The source uses it to argue that perfectionism is not a badge of seriousness but a habit system that can create procrastination, fear of judgment, over-preparation, and avoidance.

In the episode's reading, the book's value is practical rather than diagnostic. It gives [[QinZong|秦总]] a vocabulary for lowering action standards, changing self-talk, treating failure as survivable, and building small proofs that action can happen before confidence or ideal conditions arrive.
