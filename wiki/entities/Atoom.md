# Atoom

Atoom is [[Edward]]'s team's software-track second-prize product from the [[XiaohongshuHackathonPeakCompetition]] in [[ba-7-wei-heikesong-xuanshou-qing-jin-boke-guanjun-guai-cai-he-48-xiaoshi-bumian-de-yexinjia-lhozhsuqbw8csa5tj5tqc7saqrex]]. The name comes from "attention." The product aims to dynamically adjust or rebuild web pages around what the user currently needs to notice, moving relevant content and buttons into more useful positions.

The project connects the source's [[VibeCoding]] theme to interface design: once implementation gets faster, the product question becomes where attention should go and what interaction pattern is worth preserving or changing.
