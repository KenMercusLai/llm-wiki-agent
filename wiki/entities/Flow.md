# Flow

Flow is the iPhone dictation app [[ChristopherMims]] mentions in [[tech-20260202-0202-mp-tech-pod-128-tech-20260202-0202-mp-tech-pod-128]]. Mims says he no longer types text messages and instead uses Flow, which the source says relies on an open-source transcription model first created by [[OpenAI]].

In the wiki, Flow is a concrete example of [[VoiceInteraction]] moving AI from a chat box into everyday communication. It also supports the episode's [[MundaneAIUseCases]] frame: replacing typing with voice is not flashy, but it can reduce repeated friction in daily work and messaging.
