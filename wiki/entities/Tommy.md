# Tommy

Tommy is introduced in [[dang-women-zai-taolun-harness-de-shihou-women-zai-taolun-shenme-shendu-duitan-minimax-hermes-agent-lvhm1cfno7mqmfv3g0aajmw4zdpd]] as the business lead for [[HermesAgent]]. He defines Hermes Agent as an open-source agent framework and explains the product through tools, main-loop control, state management, error handling, multi-layer memory, and workflows that can be saved as [[AISkills]].

## Source Position
- Describes the agent framework as the hands around the model brain, making [[AgentHarness]] concrete through execution and state management.
- Emphasizes [[PersistentAgentMemory]] as the reason users may move from earlier domestic agent experiments toward Hermes Agent.
- Presents [[AgentSelfEvolution]] as knowledge compression: successful workflows become skills that help future runs become more consistent.
- Connects Hermes Agent's origin to an open-source Discord community that had worked on Llama post-training, context-length extension, and distributed-training experiments.
