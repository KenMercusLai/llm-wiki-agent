# Kedaibiao Lizheng

Kedaibiao Lizheng is the guest in [[gaoshou-zenme-yong-ai-putongren-zenme-xue-ai-touziren-ruhe-tou-ai-duitan-kedaibiao-lizheng-ljqyo4tz0o2-pmsl-mjx6umsuzsc]]. The source describes him as working on community, courses, and enterprise training, with a focus on helping people become effective users of AI.

## Position In The Episode
- Argues that ordinary users should stop limiting themselves to chat and begin using [[AgenticWorkflow]] tools such as [[Codex]], [[ClaudeCode]], and [[Cursor]].
- Treats [[ContextEngineering]] as the main differentiator once base models and tools become similar.
- Defines strong [[AISkills]] as explicit encodings of process, boundaries, quality standards, tool use, and required context.
