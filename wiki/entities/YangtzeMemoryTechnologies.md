# Yangtze Memory Technologies / 长江存储

Yangtze Memory Technologies is the domestic Chinese NAND company discussed in [[cunchu-sanjutou-po-wanyi-shizhi-cunchu-chaoji-zhouqi-heshi-neng-jianding-s10e13-c47ff830-8cb5-4e58-b7d7-1a04e4e5a4c1]]. The source distinguishes it from [[ChangXinMemory]] by product role: Yangtze Memory is the NAND-side company, while ChangXin is mainly tied to DRAM.

The episode says Yangtze Memory's NAND is widely sold domestically, making it part of the supply-side response to [[AgentEraNANDStorage]] and [[MemoryChipShortage]]. It does not provide detailed capacity or technology disclosures, so the page treats the company as a domestic storage supplier signal rather than a fully assessed technical peer to overseas leaders.
