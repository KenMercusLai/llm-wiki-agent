# Interaction Model

Interaction Model is the [[ThinkingMachinesLab]] model discussed in [[ai-jibao-26q2-cong-coding-dao-rsi-qiangzhe-yu-qiang-de-weilai-1-171-1]]. The source says it supports listening while speaking, seeing actions, handling interruptions, and pairing real-time interaction with an asynchronous reasoning model.

The model's importance is its interaction thesis. The episode argues that [[VoiceInteraction]] should not be treated as a minor multimodal add-on: low-latency, interruptible, full-duplex voice could become a basic interface layer for AI assistants, agents, tutors, and collaborative work.
