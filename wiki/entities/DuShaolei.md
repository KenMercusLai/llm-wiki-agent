# Du Shaolei

Du Shaolei, also called Simon in the source, is introduced in [[e242-zuikuai-bannian-ai-paotong-zi-jinhua-yu-chen-tianqiao-shouxi-kexuejia-liaoliao-guigu-moxing-bi-zheng-zhi-di]] as a University of Washington computer science associate professor and [[Apodex]] chief scientist responsible for training and inference. This page uses `DuShaolei` to avoid confusing him with the existing [[Simon]] page for a different guest.

In the episode, Du frames [[DeepResearch]], coding, and self-evolution as expressions of the same core model capabilities: long-horizon reasoning, environment exploration, tool use, and feedback-driven continuation. He also emphasizes that [[DiscoveryModel]] work needs [[ResearchTaste]], not only benchmark skill, because a scientific model must learn which questions are worth pursuing.
