# Manus

Manus is referenced in [[agent-yuannian-di-500-tian-shenme-zai-xiaoshi-shenme-zai-dansheng-weishenme-women-bugai-zai-touzi-gui-siwei-de-ruanjian-lhwdxfpke3bmamjk4e6knk-5sn-b]] as one of the important agent-era milestones during the roughly 500 days after the show's earlier "Agent Year One" episode. It is also used as an example of a newer agent service that may handle tasks such as travel discovery differently from incumbent app workflows.

[[renlei-he-ai-agent-de-zuijia-peihe-fangshi-hai-mei-bei-faming-duitan-paperboy-ltgxurpseowqggfvgc32aurymt-o]] adds Manus as a successful agent product that [[Paperboy]] compares itself against. The source still treats Manus as relatively project/session oriented, while Paperboy's desired differentiation is [[OSLevelContext]], [[PersistentAgentMemory]], and more personalized [[HumanAgentCollaboration]].

[[tan-mi-claude-code-gao-dong-agent-harness-dui-tan-lai-xin-lu-lkluk3i7c4gzw4jvxee7odsfgis3]] references Manus in the context of agent memory and sandbox design. [[LaiXinlu]] groups Manus with [[ClaudeCode]] as using a Unix-file and agent-maintained-memory direction, while contrasting traditional cloud sandboxes with [[KComputer]]'s lighter virtual-computer approach.
