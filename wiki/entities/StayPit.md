# StayPit

StayPit is described in [[vol-161-cong-kaifa-ziji-de-openclaw-liaoqi-1-6626-1]] as the author associated with [[OpenClaw]]. The episode says [[JustinYan]] noticed StayPit's earlier Apple-ecosystem and small-tool work before OpenClaw became visible in Chinese agent discussions.

## Source Position
- StayPit is important in this source mainly as the origin point for the OpenClaw project that triggered the hosts' broader agent discussion.
- The project's naming path from ClawBot to MoteClaw to OpenClaw is used to situate it near the Claude/Anthropic naming ecosystem without treating it as an Anthropic product.
- The source uses StayPit's work to explain why personal agents became interesting to builders, not just end users.

## Connections
- [[OpenClaw]] — project associated with StayPit in the episode.
- [[JustinYan]] — host who followed StayPit's earlier tool work.
- [[AgentNativeSoftware]] and [[AgentHarness]] — concepts surfaced through the OpenClaw build pattern.
