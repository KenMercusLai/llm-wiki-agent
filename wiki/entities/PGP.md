# PGP

PGP appears in [[ba-ai-chuicheng-hewuqi-de-ren-qinshou-laxiale-xinlengzhan-tiemu-1]] as a historical analogy for software export-control failure. The hosts recall the case where encryption source code could be printed and circulated as text, using it to argue that [[AIExportControls]] over model code, weights, prompts, and API-mediated capability are harder to enforce than controls over physical goods.

## Source Position
- PGP is not the episode's main AI topic; it is an analogy for intangible information products.
- The hosts use the example to distinguish software/source-code circulation from chip or hardware shipment.
