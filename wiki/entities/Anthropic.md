# Anthropic

Anthropic is discussed in [[gaoshou-zenme-yong-ai-putongren-zenme-xue-ai-touziren-ruhe-tou-ai-duitan-kedaibiao-lizheng-ljqyo4tz0o2-pmsl-mjx6umsuzsc]] alongside [[OpenAI]]. The source references Anthropic in relation to frontier model economics, Claude-based agent tools, and [[ForwardDeployedEngineer]] roles for enterprise AI adoption. [[duihua-minimax-yan-junjie-m3-10x-jihua-10t-moxing-he-zhineng-de-zhongju-lqtilt8flvmv99v0gshhyfyraibe]] adds Anthropic as an example of a frontier AI company that brings economists, psychologists, nuclear physicists, philosophers, and other domain experts into AI work. [[na-tiao-luxian-caineng-tongwang-shijie-moxing-de-zhongju-duihua-huang-biwei-aether-ai-chuangshiren-lgg-env6jrpgvyiwtxw6bocdzdmr]] mentions Anthropic as another frontier lab that [[HuangBiwei]] sees as mostly staying within the LLM paradigm rather than fully moving toward [[CausalAI]]. [[ep88-chuanyue-lianghua-zhifu-ximengsi-ai-hui-rang-putongren-geng-rongyi-zhuanqian-haishi-geng-nan-lhvigzza2ugmayezkbrxufkmp4l1]] adds Anthropic to the [[AIIPOValuation]] discussion as an example of a private AI company where public-market investors must separate technological importance from entry price. [[openai-he-anthropic-gongtong-kanhao-de-fde-ai-shidai-de-xin-gangwei-chuxian-jiu-fengong-songdong-duitan-rolling-ai-ljlatrjimrlnbe-luqmat0c74xo6]] adds that Anthropic and [[OpenAI]] are discussed as emphasizing FDE-style enterprise deployment because industry transformation requires deep business context, workflow change, and human-AI operating design. [[renlei-he-ai-agent-de-zuijia-peihe-fangshi-hai-mei-bei-faming-duitan-paperboy-ltgxurpseowqggfvgc32aurymt-o]] adds Anthropic as a key competitive force for [[Paperboy]] and as the company context behind [[ClaudeCode]] and Opus-quality agent work.

[[tan-mi-claude-code-gao-dong-agent-harness-dui-tan-lai-xin-lu-lkluk3i7c4gzw4jvxee7odsfgis3]] adds Anthropic through [[ClaudeCode]] and managed-agent/harness capability. [[LaiXinlu]] treats Anthropic's model quality as the first source of agent performance, while using Claude Code to argue that strong models still need well-aligned [[AgentHarness]] design around context, memory, tools, permissions, and handoff.

[[dang-women-zai-taolun-harness-de-shihou-women-zai-taolun-shenme-shendu-duitan-minimax-hermes-agent-lvhm1cfno7mqmfv3g0aajmw4zdpd]] adds Anthropic through Claude Code real-name discussion, Opus-quality agent behavior, and source-leakage analysis. The source uses Anthropic as both a benchmark for agent capability and a case where [[AgentIdentityAndAuthentication]], safety narratives, and ecosystem openness come into tension.

[[ep108-vibe-coding-da-dizhen-cursor-dingjia-zhengyi-windsurf-shougou-fengbo-moxing-changshang-qin-erzi-men-you-jiang-ruhe-jinchang-lqn-icq1xqgk7xxxxzrpunj4fan]] adds Anthropic through [[ClaudeCode]] as an official coding tool that can compete directly with [[Cursor]] once Cursor's pricing feels closer to model API economics. The episode treats Anthropic's tool as evidence that model providers will not leave the highest-token, easiest-to-verify coding workflow entirely to startups.

[[vol-166-xianliao-cong-gemini-dao-ai-de-jiasu-yu-hundun-1-6650-1]] adds Anthropic through the hosts' route comparison among [[Anthropic]], [[OpenAI]], and [[Gemini]]. Anthropic is described as especially focused on coding through [[ClaudeCode]], whose planning and agent behavior make it central to the source's [[Superpowers]] and [[SubagentWorkflow]] discussion.
