# GLM5

GLM5 is the [[ZhipuAI]] model named in [[zhili-bianzhi-de-chunjie-jianwenlu-yu-nachang-zhengzai-yunniang-de-youdai-weiji-1]] as part of the host's Spring Festival AI-coding experiments. The source uses it less as a benchmark story and more as a hands-on example of how coding has become cheap enough for one person to build websites, company pages, iOS apps, podcast-editing tools, and small business tools quickly.

## Key Points
- The episode's GLM5 examples support [[AIProgrammingEngineShift]]: writing code is no longer the only or main bottleneck.
- Deployment, App Store Connect, privacy policies, pricing, filing, review, operations, and platform know-how remain hard even when the model writes usable code.
- The model is used to support [[IntelligenceDevaluation]] for technical labor, but the episode still treats product judgment and domain know-how as scarce.
