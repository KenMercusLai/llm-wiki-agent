# React Native

React Native appears in [[tsr-s3-replit-v2-tsr-s3-replit-v2]] through [[AmjadMasad]]'s [[Facebook]] chapter. After [[Codecademy]], Amjad joined Facebook because he wanted a harder technical challenge and worked on the early React Native team, which he describes as startup-like inside a large company.

The source uses React Native mainly as a career and technical-ambition step before [[Replit]] became Amjad and [[HayaOdeh]]'s full company. Amjad even considered turning Replit into a Facebook project, but worried he would lose ownership and end up with a boss. That makes React Native part of the episode's [[FounderControl]] and [[ConsumerGradeDeveloperInfrastructure]] background rather than the central product.
